# AppTypeScript
Módulo: DWES

Descripción: Es una ejemplo de app desarrollada con TypeScript. 
Se basa en un sistema de regantes que contiene a dos tipos de personas, los propietarios y los arrendatarios. Según el tipo de persona varía el importe que se paga por parcela.

Última actualización: 24/11/2021

Autor: Rodrigo Robles Miñambres

Funcionalidades:

    - Creación de personas.
    - Eliminación de personas.
    - Visualización de la lista de personas.
    - Visualización del importe absoluto del sistema.
  
Front End

    HTML
    CSS
    TypeScript
    JavaScript

Back End

     Node.js 17.18.0
